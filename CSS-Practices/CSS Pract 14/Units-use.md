When to Use Each Unit
1. Percentage (%)
Ideal for fluid layouts where element sizes should adapt to their parent container.
Commonly used for width and padding.

2. em
Use when you need to scale elements relative to the current font size.
Useful for nested elements where you want styles to scale consistently.

3. rem
Suitable for consistent scaling across the document.
Recommended for global font sizes and spacing.

4. Pixels (px)
Use for fixed sizes that need to remain constant.
Ideal for borders, small icons, or elements with specific size requirements.

5. Viewport Height (vh) and Viewport Width (vw)
Perfect for full-screen layouts or elements that should scale with the viewport size.
Useful for setting background images, sections, or banners that span the entire screen.
